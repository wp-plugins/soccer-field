== Description ==

When called this plugin shows a Soccer Field inside your post.

Plugin Features:

- The easy way to embed a soccer field in your posts.
- Customizable colors

== Installation ==

1. Upload the Soccer Field plugin to your blog and activate it. 
2. Customize the plugin through: Setting -> Soccer Field
